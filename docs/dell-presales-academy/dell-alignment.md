# Dell Alignment Notes

## Purpose Of This Document

This document explains how the TCO Repatriation Dashboard can be framed for Dell Presales Academy recruiters.

It does not claim the project is an official Dell Technologies product.

## Public Dell Themes Used

The project was aligned against public Dell/Dell Technologies messaging available as of May 30, 2026.

### 1. Technology Should Drive Practical Progress

Dell describes its purpose around driving human progress through technology.

Project alignment:

- The dashboard helps make infrastructure decisions more understandable.
- It turns technical cost data into a business decision.
- It supports better customer outcomes through transparent analysis.

Recruiter framing:

```text
I tried to build a tool that uses technology to make a customer decision clearer, not just a tool that shows technical features.
```

### 2. Simplified Technology Consumption

Dell APEX messaging emphasizes simplified consumption and as-a-Service models across infrastructure.

Project alignment:

- The local infrastructure option is modeled as a monthly subscription.
- The comparison uses OPEX-style thinking instead of only hardware purchase cost.
- The dashboard helps customers understand predictable consumption over time.

Recruiter framing:

```text
The model reflects the way many customers want to consume infrastructure today: with predictable cost, subscription thinking, and simplified decision-making.
```

### 3. Hybrid, Multicloud, Data Center, And Edge Reality

Dell public messaging discusses modern workloads across on-premises data centers, cloud, and edge environments.

Project alignment:

- The recommender avoids binary cloud versus on-premises thinking.
- It frames the decision as workload placement.
- It supports hybrid recommendations when the economics are mixed.

Recruiter framing:

```text
I designed the recommendation to be hybrid by default because that is how real infrastructure decisions usually work.
```

### 4. Enterprise AI Infrastructure Economics

Dell AI infrastructure messaging highlights that cost, security, and data challenges can slow AI progress.

Project alignment:

- The same TCO discipline can be applied to AI and data-heavy workloads.
- The dashboard helps explain why some predictable or data-intensive workloads may need local infrastructure evaluation.
- It keeps security and operational considerations as validation points rather than pretending cost alone decides everything.

Recruiter framing:

```text
AI makes infrastructure placement even more important because cost, data gravity, and security become harder to ignore.
```

## How To Say It In An Interview

Use this:

```text
I aligned the project with Dell's public direction around hybrid infrastructure, simplified consumption, modern data centers, edge, and AI infrastructure economics. The dashboard is not a Dell product, but it demonstrates that I understand the customer conversations Dell is positioned to lead.
```

## Boundaries

Do not say:

```text
This is a Dell solution.
```

Say:

```text
This is a candidate project aligned to the types of customer problems Dell helps solve.
```

Do not say:

```text
This proves Dell is cheaper than cloud.
```

Say:

```text
This helps structure a workload placement and TCO discussion.
```
