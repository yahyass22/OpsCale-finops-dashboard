# Dell Presales Academy Presentation Package

This folder contains a recruiter-facing presentation package for the TCO Repatriation Dashboard.

Audience:

- Dell Presales Academy recruiters.
- Interviewers for an Inside Product Solution Architect role.
- Technical or business reviewers who want to see presales thinking, not only coding ability.

Positioning:

```text
This is a candidate-built presales tool that turns infrastructure cost pressure into a structured customer conversation about workload placement, hybrid strategy, and business value.
```

## Files

| File | Purpose |
| --- | --- |
| [presentation-deck.md](presentation-deck.md) | Slide-by-slide presentation with speaker notes |
| [one-page-brief.md](one-page-brief.md) | Recruiter-friendly one-page project summary |
| [role-alignment.md](role-alignment.md) | How the project maps to an Inside Product Solution Architect role |
| [demo-runbook.md](demo-runbook.md) | Live demo flow for a 7-10 minute recruiter presentation |
| [qa-prep.md](qa-prep.md) | Likely interview questions and strong answers |
| [dell-alignment.md](dell-alignment.md) | Dell public-message alignment and source notes |
| [talk-track.md](talk-track.md) | Short, medium, and long verbal pitches |

## How To Use This Package

1. Read [one-page-brief.md](one-page-brief.md) first.
2. Practice the 90-second pitch in [talk-track.md](talk-track.md).
3. Use [presentation-deck.md](presentation-deck.md) if they ask you to present the project formally.
4. Use [demo-runbook.md](demo-runbook.md) when showing the live app.
5. Review [qa-prep.md](qa-prep.md) before interviews.

## Important Boundary

This is not an official Dell Technologies tool, architecture, or recommendation engine. It is a candidate project designed to demonstrate presales reasoning, business value framing, technical architecture, and customer-facing communication.
